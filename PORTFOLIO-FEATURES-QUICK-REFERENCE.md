# Portfolio Management - Quick Feature Reference

## 🎯 New Features at a Glance

### 1. Purpose Section
**Location:** Top of page, after hero
**What it does:** Explains why portfolio management matters
**Key info:** 4 value propositions + benefits list + impact statement

### 2. Dynamic Health Dashboard
**Location:** Below purpose section
**What it does:** Real-time portfolio health visualization
**Views:** 
- Health Overview (bar charts by category)
- Health Dimensions (4 scoring dimensions with tooltips)
**Metrics:** 601 total assets tracked across 4 categories

### 3. Key Metrics Dashboard
**Location:** Below health dashboard
**What it does:** At-a-glance portfolio statistics
**Metrics:**
- Primary: 247 apps, 74% health, $42.3M spend, 23 retires
- Secondary: 89 cloud ready, 62% tech debt, 91% security, $3.2M savings

### 4. Compliance Alerts
**Location:** Middle of page
**What it does:** Track certifications, renewals, EOL, security, audits
**Features:**
- Filter by severity (Critical/High/Medium)
- Filter by type (5 alert types)
- Summary cards (Critical, Overdue, Next 30 days, In Progress)
- Days until due/overdue calculations

### 5. Dependency Visualization
**Location:** Below compliance alerts
**What it does:** Map relationships between assets
**Views:**
- Dependency Map (visual)
- List View (tabular)
**Shows:** Upstream/downstream dependencies + impact assessment

### 6. Enhanced Search
**Location:** In main content area
**Features:**
- Search suggestions
- Recent searches
- Saved searches (3 pre-configured)
- Active filter badges
- One-click filter removal

### 7. DTMP Integration
**Location:** Bottom of page, before footer
**What it does:** Shows integration with other marketplaces
**Integrations:**
- Lifecycle Management
- Digital Intelligence
- Governance & Compliance

## 🎨 Color Coding Guide

### Health Status:
- 🟢 Green (71-100%): Healthy
- 🟡 Yellow (41-70%): Watch
- 🔴 Red (0-40%): At Risk

### Severity Levels:
- 🔴 Critical: Immediate action required
- 🟠 High: Urgent attention needed
- 🟡 Medium: Plan for resolution

### Asset Types:
- 🔵 Blue: Applications
- 🟣 Purple: Infrastructure
- 🟢 Green: Data Assets
- 🟠 Orange: Vendor Services
- 🌸 Pink: Business Capabilities
